Navigation

Scroll to zoom in and out. Click and drag to pan. Double-click to zoom into a specific area. Pinch to zoom on touch devices.

Search

Type in the search box (or press Ctrl+K) to find assets by name or code. Use arrow keys to navigate results, press Enter to jump. You can also type coordinates (e.g. 9.05, 7.49) to fly to a location.

Layers

Switch to the Layers tab above to toggle asset types on or off. The count badge shows how many features are in each layer. The Visible Features total updates as you toggle.

Measure Distance

Click Measure in the toolbar, then click successive points on the map to draw a path. A running total distance is shown. Double-click to finish the measurement.

Installation Planning

Click Plan, then click any spot on the map. The system finds the nearest FDH cabinets, lets you pick one, then choose Auto Route (follows existing fiber) or Trace Route (click a custom path). A cost estimate is calculated from distance, materials, and labor rates.

Edit Mode

Click Edit to unlock all markers for drag-and-drop repositioning. Drag a marker to its new location; the updated coordinates save automatically. Click Edit again (or press Esc) to lock markers back in place.

Keyboard Shortcuts

Esc — Cancel active mode (measure, plan, or edit)

Ctrl+K / ⌘K — Focus the search box

+ / − — Zoom in / out

Popups & Details

Click any marker or cable to open a popup with its name, code, and metadata. Where available, a View Details link opens the full asset page for editing.
