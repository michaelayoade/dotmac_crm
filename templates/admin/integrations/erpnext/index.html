ERPNext Import

One-time import of data from ERPNext/Frappe

Import Order

Data will be imported in the following order to preserve relationships:

  1. Contacts - Creates Person records
  2. Customers - Creates Organization + Subscriber records
  3. Projects - Links to Subscribers
  4. Tasks - Links to Projects
  5. HD Tickets - Links to Subscribers/Contacts
  6. Leads - Creates Person + Lead records
  7. Quotations - Creates Quote records with line items
